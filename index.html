Documentation

See the #more-downloads channel in the Discord server (link above) to make sure you have all required software installed. If you don't Era WILL NOT WORK.

1. Download the Era Launcher from the #downloads channel in the Discord. Be sure to download the latest version.

2. Download the version of Fortnite that you would like to play. You can find these builds in the #fn-builds channel in the Discord.

NOTE: You may need to whitelist Era from your anti-virus software. Software such as Windows Defender (the build-in Windows anti-virus app) will most likely tell you that Era is a virus. Don't worry, it's not. This is becuase the Era launcher has been obfuscated, which is a way of preventing this code being stolen. Don't fully disable your anti-virus software, but simply whitelist Era Launcher.

3. Copy the file path of the version of Fortnite you would like to play. This path should end with "/Fortnite"

4. Paste this path into the Era Launcher. Lauch it.

5. At the login screen, enter anything. As long as something is entered into both boxes, you will be able to login.

NOTE: Login can sometimes fail. If you get an error when trying to login, see the #troubleshooting channel in the Discord.

6. Once in the lobby, load into a match. It will take a minutes, but you will spawn on spawn island!

7. After a couple minutes, the battle bus will launch! If you don't want to wait that long, open the console by pressing the ` key (that's the key directly under ESC). Type "Startaircraft" (without the quotes). Hit ENTER and you'll be on the Battle Bus!

Basic Console Commands

Open the console by pressing ` (they key directly under ESC). Execute commands by pressing ENTER. Slomo (1-3) - increases speed, 1 is default speed
God - invincibility
toggledebugcamera - freecam
Teleport - teleports your character to wherever you are looking
Demospeed (1-10) - changes speed
Destroytarget - destroys what you are looking at
Pausesafezone - pauses storm circle
Startaircraft - starts battle bus early
